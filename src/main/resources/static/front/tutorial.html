So Funktioniert es!

Hinzufügen von Text:

- auf „Text“ klicken
- Textfeld mit einem Klick auf der Karte platzieren


Bearbeiten von Text:

- bewegen: auf das Textfeld klicken, linke Maustaste gedrückt halten, Textfeld bewegen
- schreiben: in das Textfeld klicken, Text ändern
- gestalten: Schriftart und Schriftgröße mithilfe der Werkzeugleiste ändern


Hinzufügen von Bildern:

- auf „Bild/Logo“ klicken
- Bilddatei auswählen
- Bild mit einem Klick auf der Karte platzieren


Bearbeiten von Bildern:

- bewegen: auf das Bild klicken, linke Maustaste gedrückt halten, Bild bewegen
- Größe ändern: auf das Bild klicken, dann auf eine Ecke klicken, die linke Maustaste gedrückt halten, Maus bewegen


Bewegen und Zoomen der Karte:

Karte bewegen:
- Mausrad gedrückt halten (Maustaste 3) um Karte im Editor zu bewegen
- zurück zum Anfangszustand: auf „Zentrieren“ drücken.


Karte zoomen:
- Steuerungstaste (STRG/CTRL) drücken und gleichzeitig scrollen
- zurück zum Anfangszustand: auf „Zentrieren“ drücken.


Ändern der Ansicht:

- wechseln Sie mit den Buttons rechts oben die Ansicht:
„Druckbogen“: Vorder- und Rückseite der aufgeklappten Karte
„einzelne Seiten“: die Karte kann im Editor geklappt werden


Nutzer-Design speichern

- wählen Sie „Server“ oder „lokal“
- klicken Sie auf „Speichern“


Laden einer Nutzer-Design Datei


erstellen der Druckdatei

- auf „PDF erzeugen“ klicken
- die Druckdatei wird an Kettcards gesand
